## üì± Project Tasks

I have successfully completed the following tasks as part of the internship requirements:

### ‚úÖ [Task 1: Language Translation Tool](./CodeAlpha_LanguageTranslationTool)
A streamlined, web-based application for translating text between multiple languages. 
- **Features**: Real-time translation, auto-language detection, Text-to-Speech (TTS), and clipboard support.
- **Tech Stack**: Python, Streamlit, `deep-translator`, `gTTS`.
- **Run**: `streamlit run app.py` within the `CodeAlpha_LanguageTranslationTool` directory.

### ‚úÖ [Task 2: Chatbot for FAQs](./CodeAlpha_FAQChatbot)
An intelligent FAQ assistant that uses NLP to match user queries with relevant answers.
- **Features**: TF-IDF & Cosine Similarity for query matching, NLTK-based text processing, and a modern Flask-based web chat interface.
- **Tech Stack**: Python, Flask, NLTK, Scikit-learn.
- **Run**: `python app.py` within the `CodeAlpha_FAQChatbot` directory.

### ‚úÖ [Task 3: Music Generation with AI](./CodeAlpha_MusicGeneration)
A deep learning model that generates new music sequences based on MIDI datasets.
- **Features**: MIDI data preprocessing, LSTM-based RNN architecture for pattern learning, and MIDI output generation.
- **Tech Stack**: Python, `music21`, Keras/TensorFlow.
- **Run**: `python app.py` within the `CodeAlpha_MusicGeneration` directory.

### ‚úÖ [Task 4: Object Detection and Tracking](./CodeAlpha_ObjectDetection)
A real-time vision system that identifies and tracks objects in a live video stream.
- **Features**: YOLOv8 (Nano) for high-performance detection, real-time object tracking with unique IDs, and OpenCV visualization.
- **Tech Stack**: Python, YOLOv8 (Ultralytics), OpenCV.
- **Run**: `python app.py` within the `CodeAlpha_ObjectDetection` directory.
